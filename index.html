Minesweeper

Minesweeper is a CLI implementation of the classical Minesweeper game fully written in C.

Download

Feel free to download the binaries under the latest release and run it via your Terminal: 

>./minesweeper.exe

Or in Linux: 

$ ./minesweeper.out
